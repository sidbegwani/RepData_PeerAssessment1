## Important Info

- The original data set for this analysis was downloaded from the following link [Link to data set](https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2Factivity.zip)

- The file was unzipped to obtain the activity.csv file

- R version 3.1.2 (32 bit) has been used to perform the analysis. The operating system was Windows 7 on an Intel processor.

- Besides the base packages, the additional R packages used are *knitr v1.6*, *reshape2 v1.4*, *lubridate v1.3.3*, and *lattice v0.20-29*

- The figures generated are placed in the figure directory of the repository
